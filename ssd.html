About

Kool OS is a modern Linux distribution with two editions: a Fedora-based distribution. Powered by KDE Plasma, designed with a clean Android-inspired UI, and made for developers and enthusiasts alike.

Features

  • Fedora Edition: Stable and up-to-date Fedora base
  • Modern Android-inspired UI with dark mode
  • Custom ISOs for each update
  • Multiple browsers preinstalled
